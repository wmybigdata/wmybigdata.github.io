虚拟机如何扩容问题

这个问题我遇到了很久，就是我们在工作的过程当中，难免会遇到自己的虚拟机出现容量不够的情况，所以我根据网上的案例自己去尝试一下是否可以扩容成功！！！

1、查看虚拟机的容量使用状况

[root@flink07 ~]# df -h

devtmpfs                 476M     0  476M    0% /dev
tmpfs                    488M     0  488M    0% /dev/shm
tmpfs                    488M  7.7M  480M    2% /run

tmpfs                     98M     0   98M    0% /run/user/0
